
# Parking Lot System – Clean Design

## Design Patterns Used
- **Singleton** – ParkingLotManager
- **Factory** – VehicleFactory
- **Strategy** – FeeCalculationStrategy
- **Repository** – TicketRepository
- **Template Method** – ParkingFlow
- Thread-safe via locks

## Features
- Vehicle-based slot allocation
- Exactly-once ticket generation
- Dynamic pricing
